# Google Play Publishing Guide

## 1. Create Play Console App

1. Open Google Play Console.
2. Create app and fill basic metadata.
3. Select category and contact details.

## 2. Complete Store Listing

Prepare:

- App title and short description
- Full description
- Feature graphic
- App icon
- Screenshots (phone required, tablet optional)
- Privacy policy URL

## 3. App Content and Policy Declarations

Complete required forms:

- App access
- Data safety
- Ads declaration
- Content rating questionnaire
- Target audience

## 4. Upload Build

1. Go to Testing > Internal testing.
2. Create release.
3. Upload `app-release.aab`.
4. Add release notes.
5. Save and roll out to internal testers.

## 5. Validate Internal Test

- Install app via Play testing link.
- Re-run smoke checklist.
- Confirm external link opening works.
- Confirm search flow works offline with bundled assets.

## 6. Promote to Production

1. Create production release from tested artifact.
2. Re-check policy warnings.
3. Submit for review.
4. Monitor review status and policy messages.

## 7. Post-Release

- Track crashes/ANRs.
- Track search usage and user feedback.
- Prepare hotfix release if critical issues are found.
